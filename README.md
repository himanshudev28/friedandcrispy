# 🍗 Fried & Crispy

A modern restaurant management and food ordering platform built with React, TypeScript, Tailwind CSS, and Supabase. The application helps restaurants efficiently manage orders, payments, and business operations through an intuitive and responsive dashboard.

## 🌐 Live Demo

https://friedandcrispy.vercel.app/

---

## 📖 Overview

Fried & Crispy is a full-stack restaurant management solution designed to streamline restaurant operations and improve customer experience. The platform provides a clean and responsive interface for managing orders, tracking payments, and monitoring business performance.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/DarkDevil-x/friedandcrispy.git
```

### Navigate to the Project

```bash
cd friedandcrispy
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
